# Academic Research Profile

A zero-build GitHub Pages academic homepage based on the MIT-licensed
[AcadHomepage](https://github.com/RayeRen/acad-homepage.github.io) template.

This version keeps the template's proven markup, responsive layout, SEO support,
and academic content model while adding a restrained Liquid Glass visual layer.
`index.html` is deployable directly, so Ruby and Jekyll are not required.

## Personalization

- Update identity, links, biography, publications, and experience in `index.html`.
- Adjust Liquid Glass styling in `assets/css/glass.css`.

## GitHub Pages

This repository is intended to be published as
[`segyulee.github.io`](https://segyulee.github.io/) from the `main` branch.
The site is plain static HTML/CSS/JavaScript and requires no build step.
